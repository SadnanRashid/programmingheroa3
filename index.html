Flexbox was designed for layout in one dimension - either a row or a column. Grid was designed for two-dimensional layout - rows, and columns at the same time. CSS Grid focuses on precise content placement. Flexbox focuses on content flow rather than content placement. Flexbox enables you to allocate space and align items flexibly.

Bootstrap comes more with responsive layouts, components, and utility classes compared to Tailwind. But Tailwind has a lot more utility classes than Bootstrap. Bootstrap is less flexible than Tailwind. Bootstrap has been around much longer than Tailwind and is one of the most popular frameworks but Tailwind is also gaining large attention in the dev community.

In CSS, the term "box model" is used when talking about design and layout. It is essentially a box that wraps around every HTML element. Every box is composed of four parts: content edge, padding edge, border edge, and margin edge.

Semantic HTML elements are those that clearly state their meaning in a readable way. Such as header, footer, article, section etc. This tags acts same as div tags. But are easy to identify their meaning. Like, header tag is used to state the header element. The article tag could be replaced with a div tag, but using article tag is easier to understand it's use.
